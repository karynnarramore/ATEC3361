The Box Model is the model that browsers use to apply CSS to your HTML markup.

You can think of it as literally a box a that wraps around each element, and it's made of content, padding, border, and margin.

{ box-sizing: content-box; }

This is the default

By default, browsers calulate the width of an element to include

  • Content

The following properties of the element are not considered part of width

  • Padding
  • Border
  • Margin

This makes it difficult to work with multi-column layouts, as you've probably noticed if you've ever tried to approach the three column problem

However, there are alternate ways to calulate the width of an element.

{ box-sizing: border-box; }

Bootstrap uses this

Setting the box sizing mode to border-box causes the browser to include

  • Content
  • Padding
  • Border

This way of doing the calculation often makes more sense to people, as it leaves the only property that is visually outside of the box out of the calulation

  • Margin
